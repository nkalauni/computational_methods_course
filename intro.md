# Computational Methods for Earth Sciences

Welcome to the graduate-level course on **Computational Methods for Earth Sciences**!

## Course Overview

This course provides an introduction to computational methods and tools used in modern earth sciences research. Students will learn programming fundamentals, data analysis techniques, numerical methods, and scientific computing best practices.

## Course Structure

This JupyterBook contains all course materials organized into the following sections:

- **[Notebooks](notebooks/index.md)**: Interactive Jupyter notebooks with code examples and exercises
- **[Slides](slides/index.md)**: Lecture slides and presentations
- **[Resources](resources/index.md)**: Additional reading materials, documentation, and reference guides
- **[Data](data/index.md)**: Datasets used throughout the course
- **[Extra Materials](extras/index.md)**: Supplementary materials, tutorials, and advanced topics

## Getting Started

To get the most out of this course:

1. Follow the notebooks in order, working through the examples
2. Complete the exercises at the end of each notebook
3. Refer to the resources section for additional learning materials
4. Use the datasets provided to practice your skills

## Prerequisites

- Basic understanding of earth sciences concepts
- Familiarity with programming (preferred but not required)
- Access to a computer with Python installed
