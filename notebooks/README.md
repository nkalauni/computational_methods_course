# Notebooks Directory

This directory contains Jupyter notebooks for the course. Notebooks should be placed here organized by topic or week.

## Organization

Suggested organization structure:
- `01_introduction/` - Introduction notebooks
- `02_python_basics/` - Python fundamentals
- `03_data_analysis/` - Data analysis notebooks
- `04_visualization/` - Visualization examples
- etc.

## Adding New Notebooks

When adding new notebooks:
1. Include clear markdown explanations
2. Add exercise sections
3. Include sample outputs
4. Test the notebook before committing
